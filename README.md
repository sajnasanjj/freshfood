Fresh food - Food delivery application

web application for delivery food based on one restuarant

(Features includes like e commerce site) .

Fresh food allows users to find their favorite food based on a restaurant and make deliver online
. OTP Verification and log in using Twilio.
. Offers and Coupon
. Created an admin side
. Admin can manage foods, categories, offer, coupon and orders. They also can manage users.
. Admin can manage and download sales reports.
. User can manage their cart, orders wish list and payment.
. Payment gateways integrated: Razorpay, PayPal
. Hosted the website on AWS using Nginx
. Load balancer



Technologies used:

Node.js
Express.js
MongoDB
Handlebars
AWS
Nginx
EC2
Figma
Bootstrap
